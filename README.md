# ddungdaeng_frontend
React-Native 기반 반려견 건강관리 앱 프론트엔드

## 🛠 기술 스택
- React Native
- TypeScript
- Recoil or Zustand (선택)
- Axios

## 📂 브랜치 전략
- main: 배포 브랜치
- dev: 통합 개발 브랜치
- feature/*: 기능 단위 작업 브랜치

## ✅ 커밋 컨벤션
- feat: 기능 추가
- fix: 버그 수정
- chore: 설정, 빌드
- docs: 문서 변경
- refactor: 리팩토링
예: `feat: 그룹 페이지 UI 작성`

## ⛳ 협업 규칙
- `dev` 기준으로 PR → 리뷰 후 `main`에 병합
- PR 제목: 작업 목적 명확하게
- 기능 구현 후 반드시 `리뷰 → merge` 절차
